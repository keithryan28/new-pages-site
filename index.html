JavaScript Variable Exercises

Exercise 1: Declare and Assign a Variable

Declare a variable named myName and assign your name to it as a string. Print the value of myName to the console.

let myName = "John";
console.log(myName);
Exercise 2: Use const for Constant Values

Declare a constant variable named pi and assign it the value 3.14. Print the value of pi to the console.

const pi = 3.14;
console.log(pi);
Exercise 3: Reassign a let Variable

Declare a variable named myAge and assign it the value 20. Reassign the value of myAge to 25. Print the new value of myAge to the console.

let myAge = 20;
myAge = 25;
console.log(myAge);
Exercise 4: Declare Multiple Variables

Declare three variables: firstName, lastName, and fullName. Assign your first and last names to the respective variables, and concatenate them to assign to fullName. Print fullName to the console.

let firstName = "John", lastName = "Doe";
let fullName = firstName + " " + lastName;
console.log(fullName);
Exercise 5: Use Different Data Types

Declare a variable named isLearning and assign it the boolean value true. Declare a variable named score and assign it the number 100. Print both variables to the console.

let isLearning = true;
let score = 100;
console.log(isLearning);
console.log(score);
Exercise 6: Combine Strings and Numbers

Declare a variable named message and assign it the value "You scored " + score + " points!". Use the score variable from the previous exercise. Print the value of message to the console.

let score = 100;
let message = "You scored " + score + " points!";
console.log(message);
