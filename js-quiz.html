JavaScript Function Exercises

Exercise 1: Write a Simple Function

Write a function named greet that takes no parameters and logs "Hello, World!" to the console. Call the function.

function greet() {
  console.log("Hello, World!");
}
greet();

Exercise 2: Use Parameters in a Function

Write a function named sayHello that takes a single parameter name and logs "Hello, [name]!" to the console. Call the function with your name as the argument.

function sayHello(name) {
  console.log("Hello, " + name + "!");
}
sayHello("Alice");

Exercise 3: Return a Value from a Function

Write a function named addNumbers that takes two parameters a and b. The function should return the sum of the two numbers. Call the function and store the result in a variable, then log the result.

function addNumbers(a, b) {
  return a + b;
}
let result = addNumbers(5, 10);
console.log(result);

Exercise 4: Use Default Parameters

Write a function named multiply that takes two parameters a and b. If b is not provided, it should default to 1. The function should return the product of the two numbers. Call the function with and without the second argument.

function multiply(a, b = 1) {
  return a * b;
}
console.log(multiply(5, 3)); // 15
console.log(multiply(7));    // 7

Exercise 5: Function Naming Conventions

Why is it important to use meaningful and descriptive names for functions? Write a short function named calculateArea that takes a single parameter radius and returns the area of a circle. Use the formula: area = π * radius * radius (use Math.PI for π).

function calculateArea(radius) {
  return Math.PI * radius * radius;
}
console.log(calculateArea(5)); // 78.53981633974483

Exercise 10: Function Scope

Write a function named testScope that declares a variable message inside it and assigns the value "This is local". Log the variable inside the function and attempt to log it outside the function. Explain the result.

function testScope() {
  let message = "This is local";
  console.log(message); // Logs: "This is local"
}
testScope();
console.log(message); // Error: message is not defined (because it's scoped to the function)
