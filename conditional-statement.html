JavaScript If Statement Exercises

Exercise 1: Basic If Statement

Write an if statement that checks if the variable number is greater than 10. If it is, log "The number is greater than 10" to the console.

let number = 15;
if (number > 10) {
  console.log("The number is greater than 10");
}
Exercise 2: If-Else Statement

Write an if-else statement that checks if a variable age is at least 18. If it is, log "You are an adult" to the console. Otherwise, log "You are a minor".

let age = 16;
if (age >= 18) {
  console.log("You are an adult");
} else {
  console.log("You are a minor");
}
Exercise 3: Multiple Conditions

Write an if-else if-else statement that checks a variable grade and logs:

  • "A" if the grade is 90 or above
  • "B" if the grade is between 80 and 89
  • "C" if the grade is between 70 and 79
  • "Fail" if the grade is below 70
let grade = 85;
if (grade >= 90) {
  console.log("A");
} else if (grade >= 80) {
  console.log("B");
} else if (grade >= 70) {
  console.log("C");
} else {
  console.log("Fail");
}
Exercise 4: Checking Equality

Write an if statement that checks if a variable username is equal to the string "admin". If it is, log "Welcome, admin" to the console.

let username = "admin";
if (username === "admin") {
  console.log("Welcome, admin");
}
Exercise 5: Nested If Statements

Write a nested if statement to check if a variable number is positive. If it is, check if the number is even or odd. Log "Positive and even" or "Positive and odd" accordingly.

let number = 7;
if (number > 0) {
  if (number % 2 === 0) {
    console.log("Positive and even");
  } else {
    console.log("Positive and odd");
  }
}
